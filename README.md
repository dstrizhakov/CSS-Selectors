## CSS-Selectors game 

This is a clone of famous css dinner game (https://flukeout.github.io/)

# [deploy](https://dstrizhakov.github.io/CSS-Selectors/index.html)

### Using technologies:

- TypeScript
- Webpack
- ESlint
- Prettier

### How to install:

```sh
cd CSS-Selectors
npm i
npm start
```
